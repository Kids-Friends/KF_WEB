# 🤖 Kids-Friends 로봇 관리자 시스템 (Admin Dashboard)

안녕하세요! 이 프로젝트는 **키즈카페에서 활동하는 'Temi 로봇'을 쉽게 관리하고 제어하기 위한 관리자용 웹사이트**입니다. 
복잡한 기술 지식 없이도 로봇의 상태를 확인하고, 손님들이 로봇과 어떻게 대화했는지 한눈에 파악할 수 있도록 만들어졌습니다.

---

## 🌟 주요 기능 (무엇을 할 수 있나요?)

1. **📊 통합 대시보드**
   - 현재 가입된 총 회원 수, 가동 중인 로봇 대수, 호출 현황을 예쁜 카드 형태로 한눈에 볼 수 있습니다.
   - 사람들이 로봇을 왜 불렀는지(호출 사유)와 어떤 대화를 나눴는지(채팅 유형)를 **그래프(차트)**로 쉽게 파악할 수 있습니다.

2. **👥 회원 및 로봇 관리**
   - **회원 관리:** 키즈카페에 등록된 회원들의 목록을 보고 포인트를 관리할 수 있습니다.
   - **로봇 관리:** 새로운 로봇을 등록하거나, 현재 로봇이 잘 작동하고 있는지(활성/오류 상태)를 끄고 켤 수 있습니다.

3. **📝 로그(기록) 확인**
   - **호출 로그:** 손님들이 로봇을 언제, 왜 불렀는지 확인하고 '처리 완료' 상태로 바꿀 수 있습니다.
   - **채팅 로그:** 아이들이 로봇과 어떤 귀여운 대화를 나눴는지, 사진을 찍었는지 기록을 볼 수 있습니다.

4. **⚙️ 쉬운 설정 (서버 주소 변경)**
   - 와이파이(Wi-Fi) 환경이 바뀌어 서버 주소가 변경되더라도, 설정 페이지에서 **IP 주소만 간단히 입력**하면 전체 시스템이 알아서 새로운 주소로 연결됩니다.

---

## 🚀 시작하는 방법 (어떻게 실행하나요?)

컴퓨터에 프로그램(Node.js)만 깔려있다면 누구나 쉽게 실행할 수 있습니다!

**1단계: 프로그램 설치하기**
화면 하단의 터미널(까만 창)에 아래 명령어를 치고 엔터(Enter)를 누르세요.
\`\`\`bash
npm install
\`\`\`
*(이 프로젝트를 실행하는 데 필요한 부품들을 다운로드하는 과정입니다. 조금만 기다려주세요!)*

**2단계: 관리자 사이트 켜기**
설치가 다 끝났다면 아래 명령어를 치고 엔터(Enter)를 누르세요.
\`\`\`bash
npm run dev
\`\`\`

**3단계: 인터넷 창에서 확인하기**
이제 구글 크롬(Chrome)이나 엣지(Edge) 같은 인터넷 창을 열고, 주소창에 아래 주소를 복사해서 붙여넣으세요!
> **http://localhost:5173/admin/dashboard**

짜잔! 예쁜 관리자 대시보드 화면이 나타날 것입니다. 🎉

## 🛠 사용된 기술 (참고용)
- **화면 만들기:** React (가장 인기 있는 웹사이트 제작 도구)
- **그래프 그리기:** Recharts (데이터를 예쁜 차트로 만들어주는 도구)
- **서버와 대화하기:** Axios (데이터베이스에서 정보를 가져오게 도와주는 도구)
