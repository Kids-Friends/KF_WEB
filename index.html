PROBLEM

왜 로봇이 필요한가?

키즈카페 현장의 반복되는 비효율을 직접 해결합니다

💬
반복 질문 응대
놀이존 위치, 이용 규칙, 요금 안내 등 동일한 질문이 반복됩니다.
🔔
잦은 직원 호출
아이 관련 사고나 요청마다 직원이 직접 이동해야 합니다.
🛡️
안전관리 집중도 저하
반복 응대 때문에 정작 중요한 안전관리와 현장 대응이 끊깁니다.

IDEA STORY

왜 키즈카페인가?

여러 시나리오 중 키즈카페를 선택한 이유

❓
반복 질문 최다 현장
아이와 보호자의 반복 질문이 가장 많이 발생하는 공간입니다.
🚶
직접 이동 안내 필요
직원이 직접 이동하며 안내해야 하는 구조적 문제가 있습니다.
⚠️
안전 집중 필요
반복 응대로 인해 정작 중요한 안전관리 집중이 어렵습니다.
🤖
Temi 최적 적합
Temi의 이동성·화면·음성·자율주행 기능과 가장 잘 맞습니다.
📊
데이터 확장 가능
질문/호출 로그를 저장해 운영 데이터로 확장할 수 있습니다.

SOLUTION

Kids-Friends Robot

Temi 로봇을 중심으로 AI 대화, 자율 이동 안내, 센서 이벤트, 관리자 대시보드를 연결한 키즈카페 운영 보조 시스템입니다.

🤝
이동형 친구 UX
고정형 키오스크가 아닌, 아이 곁으로 직접 다가가는 친구 경험
📈
운영 데이터 축적
단순 안내를 넘어 질문·호출 데이터로 운영을 최적화합니다
🔗
통합 시스템
센서 + AI + 로봇이 하나의 흐름으로 연결된 통합 플랫폼

USER SCENARIO

사용 시나리오

실제 키즈카페에서의 한 장면

1
아이가 "미끄럼틀 어디 있어?" 질문
2
Temi가 음성으로 즉시 답변
3
Temi가 해당 놀이존까지 직접 이동 안내
4
아이가 "퀴즈 해줘" 요청 → LLM 인터랙션
5
버튼 또는 화면으로 직원 호출
6
관리자 페이지에 호출 이벤트 실시간 표시
7
질문/호출 로그가 DB에 자동 저장
